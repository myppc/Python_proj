准备：
	使用前需要安装paython3
	使用到的模块有 shutil，PIL，functools，需要自己使用pip安装
	
	配置到路径文件到config.py中:
		dest_path :需要被替换的文件目录
		src_path :替换来源
		un_copy_path :未使用的图片会被拷贝进该目录

使用：
	点击 run.bat运行工具
	输出copy succ 则拷贝成功

	生成文件：			
		机器识别的非相似图片.txt		这个文件记工程中有多张同名文件，且机器对比每张文件不相似，反之则用资源文件中同名文件代替
		目标目录重复文件.txt 			工程中有多张同名文件，这些文件没有经过替换，需要肉眼比对，是否需要特殊替换
		未使用的图片.txt 				资源目录中有未使用的图片，这些图片会被拷贝进un_copy_path
		资源重复目录.txt				资源中有多张重名图片，这些图片需要美术处理，才能进行后面的替换
